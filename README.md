# Beginner Houseplant Helper

A Next.js 14 App Router website designed to assist new houseplant enthusiasts in caring for their plants. It provides a curated database of common houseplants with detailed care instructions, allowing users to build a personal "My Plants" list and track care tasks.

## Features

*   **Plant Library**: Browse a database of common houseplants.
*   **Detailed Care Guides**: Access static information on watering, light, humidity, and more for each plant.
*   **My Plants Dashboard**: Client-side storage for managing your personal collection of plants.
*   **Task Tracking**: Basic task tracking for individual plants in your collection.
*   **Stunning UI**: Dark theme, vibrant gradients, and glassmorphic elements for a modern look.

## Technologies Used

*   Next.js 14 (App Router)
*   React 18
*   Client-side state management with `useState` and `useEffect`
*   Client-side storage with `localStorage`
*   Pure CSS for styling (no Tailwind, no CSS Modules)

## Setup and Run

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd houseplant-helper
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.

4.  **Build for production:**
    ```bash
    npm run build
    ```
5.  **Start the production server:**
    ```bash
    npm run start
    ```
